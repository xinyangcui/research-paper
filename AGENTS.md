# AGENTS.md

## 文献阅读要求
- 以文献内容为依据，优先提炼研究问题、核心思路、方法设计、实验结论和局限性。
- 总结时尽量区分：作者明确说明的内容、基于文中信息的合理推断、目前无法确认的部分。
- 对论文中未明确给出的细节，不补全、不想当然，直接写“目前不清楚”或“文中没有明确说明”。

## 输出重点
- 优先回答这篇工作解决了什么问题、为什么这样设计、实验说明了什么、还存在哪些不足。
- 总结多篇文献时，突出共同点、关键差异、各自优势与局限，避免只做流水账式罗列。
- 如果涉及创新点判断，需谨慎，不将常规组合或工程实现轻易表述为“明显创新”。

## 表达要求
- 保持学术化、克制、清楚的表达。
- 结论应受文中证据约束，不夸大结果、贡献或适用范围。
- 对有争议或证据不足的判断，明确标注不确定性。
